What is web design?

Web Graphic Design

Web graphic designers are responsible for enhancing the appearance of a website using company logos, fonts and images. Designers work with the client and web developers to develop a site that promotes your company and services that you provide.

Interface Design

User interface design or UI design refers to the visual layout of the elements that a user would interact with in a website. This includes providing visual layout of a webpage that a user can easily navigate through. This includes providing clear navigation and menu options and not having too much clutter on the website.

User Experience Design

User Experience Design (UXD) is a design process whose objective is to design a system that offers a positive experience to to a website user. An effect user experience design incoroporates user interface design, usability and accessibility to all types of users. A good user experience design enhances a users experience when visiting a website.

Search Engine Optimization

Search engine optimization (SEO) is the process of improving the visibility of a website when people search for products or services related to your business in Google, Bing, and other search engines. The better visibility your website has in search results, the more likely you are to obtain visitors to your website.
